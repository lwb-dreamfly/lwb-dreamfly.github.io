初入Python之安装、破解

  俗话说，工欲善其事必先利其器，前两天做的web题目，虽然使用java成功解决，但是也费了不少劲，如果使用Python的话会好很多，所以决定要搞一搞Python了。

下载链接

  Python官网下载链接：https://www.python.org/downloads/windows/

  PyCharm官网下载链接：https://www.jetbrains.com/pycharm/download/#section=windows ，PyCharm是JeBrains打造的一款Python IDE，可以帮助我们在使用Python语言开发时提高效率，具备调试、语法高亮、智能提示等功能。要比在命令行和记事本中写程序方便的多。

  Python和Python安装很简单，一堆教程，这里就不搬运了，值得注意的是安装Python是记得把Add Python to PATH给勾选上，将python加入到Windows的环境变量中，不需要手工添加了。说一说破解过程吧。网上能够搜索到的破解教程不少，有的是有期限的，好像是一年，有的是永久有效的。我使用了一种永久有效的破解方法，一次成功，所以记录一下这个方法。先把补丁的百度云链接贴一下，其中还有上面提到的两个软件，都是官网下载的最新版，有时候官网下可能会慢一点儿。

链接：https://pan.baidu.com/s/12dGOQ6gnFG_cvEIc1njv5Q
提取码：vwa4

破解步骤
  • 在上面链接中下载补丁文件jetbrains-agent.jar 放到一个不会被轻易删除的地方，我是跟着原博主的建议放到了pycharm安装目录的bin目录下，放到哪儿都行，但是这个位置下面会用到，还是建议放到pycharm的bin目录下。
  • 进入到pycharm的工作界面，因为是第一次下载，所以点击激活窗口的“Evaluate for free”免费试用。
  • 创建一个项目Create New Project–>Create
  • 点击pycharm最上面的菜单栏中Help找到Edit Custom VM Options …，如果提示文件不存在是否要创建文件，点击Create。
  • 在打开的vmoptions编辑窗口最后添加：-javaagent:jetbrains-agent.jar的所在路径
    一定要检查好路径是否正确。
  • 修改完配置文件后重启pycharm。
  • 重启之后，在菜单栏点击Help–>Register，进入激活界面
  • 选择License server激活方式，地址填入http://jetbrains-license-server ，点击Discover Server可以自动填写。
  • 点击Active完成激活，会受到一个通知：Your copy is licensed to Administrator。
  • 也可以采用激活码激活的方式，激活码随便一搜就能找到，但是好像是有时间限制的。
