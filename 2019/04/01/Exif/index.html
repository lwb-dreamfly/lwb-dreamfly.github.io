Exif

发表于 2019-04-01 | 阅读次数：

图片隐写题，编码比较。

  经查阅得知Exif：可交换图像文件格式（Exchangeable image file format），是专门为数码相机的照片设定的，可以记录数码照片的属性信息和拍摄数据。emmmm…了解了，照片中蕴含着信息，把这些信息搞出来研究研究。

  在查阅Exif的概念时，发现了一个在线的EXIF信息查看器（https://exif.tuchong.com/ ），将图片上传，得到一堆参数 在给出的集中编码中，随便选择了比较熟悉的ASCII码尝试一下，将flag{}转换为ASCII码，然后查询匹配，最终定位到Credit这一栏。

  运气还不错，这一串数字，转化一下就是答案了。
